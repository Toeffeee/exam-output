1. How to create reactive forms?


To create reactive forms in Angular, you first need to import the ReactiveFormsModule in your application's app.component.ts file. From there, you can then define the input fields of your form represented by a FormControl inside a FormGroup.


2. How to add built-in validation for email in reactive form?


To add a built-in email validation to a form control in a reactive form, you use the Validators.email function. This function will check if the entered value is a valid email format.


3. How to add built-in validation for required fields in reactive form?


To add a built-in validation for required fields in reactive form, you use the Validators.required function. This function reminds the user that the specified field should not be left empty.


4. How to add regex in validation for reactive forms?


To add regex (or expression) in validation for reactive forms, you use the Validators.pattern function. This function is particularly useful in input fields such as usernames if you only want the user to use alphanumeric characters and no special symbols.


5. How to generate routing?


To generate routing in Angular, you can do it in the Angular CLI using the ng generate module app-routing command. Although, this is not really necessary as Angular automatically generates the routing module upon the creation of your project.


6. How to configure routes with parameters?


To configure routes with parameters, you simply add a colon (:) after your route to specify the parameter.
Example: path: 'Home', component: HomeComponent


7. What is DebounceTime in RxJS?


debounceTime is an RxJS operator that essentially adds a delay to the emission of values. This is commonly used in search bars wherein it reduces the number of unnecessary requests or actions taken while the user is still typing.


8. What is the pipe operator?


The pipe() operator is used to connect multiple RxJS operators together. An example is when you want to double a value by using the map operator and then only display the values that are greater than 10 by using the filter operator.


9. What is distinctUntilChanged operator?


The distinctUntilChanged operator basically ensures that the emitted values are different from the previous emitted value. This helps avoid duplicate values being displayed.


10. What is the filter operator in RxJS?


The filter operator is used to emit only those values that satisfy a specific condition. This helps in selectively processing emitted items that match certain criteria.
